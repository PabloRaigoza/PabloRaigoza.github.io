These set of tutorials are set to offer help others learn SDL.

The very first step to learning SDL is go ahead and download the library from the their website. The link is here: Lazy Foo' . I would suggest to go look at those tutorials as well, but I found that Lazy Foo' did a nice job at providing download information for the three primary OS. Windows, Mac, and Linux (cough...the best OS... cough).

As a little heads up before the lessons. These lesson are intened for programmers who are comfortable C++ and C. This means you can easily create classes, use constuctors, have a firm understanding of data types, and probably most important, you understand completely how to use functions and how to create them.

I to am learning more and more of SDL and the basics of Computer Science, so I might some mistakes here and there but hopefully the message can still get across with SDL
